# 🔐 Security Log Analysis Lab – Windows Authentication Monitoring

Projeto prático de Segurança da Informação com foco em SOC (Security Operations Center).

## 🎯 Objetivo

Monitorar eventos de autenticação do Windows para identificar comportamentos suspeitos, como:

- Múltiplas tentativas de login falhadas
- Logins fora do horário padrão
- Atividades anômalas de usuários

## 🛠️ Tecnologias utilizadas

- Windows Event Viewer
- Python 3
- Power BI (Free)
- VS Code

## 📂 Estrutura do Projeto

- data/raw → Logs brutos coletados do Windows
- data/processed → Dados tratados em CSV
- scripts → Scripts de coleta e análise
- alerts → Alertas gerados
- dashboards → Arquivos do Power BI
- docs → Documentação e evidências
